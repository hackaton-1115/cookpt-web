## 소개

CookPT는 냉장고에 있는 재료로 무엇을 만들지 고민하는 분들을 위한 AI 기반 레시피 생성 서비스입니다. 냉장고 사진 한 장으로 AI가 재료를 자동으로 인식하고, 원하는 국적, 테마, 조리 도구에 맞춰 **실시간으로 맞춤형 레시피**를 생성해드립니다.

## 주요 기능

### AI 재료 인식 (Google Generative AI)

- 냉장고 사진을 업로드하면 AI가 자동으로 재료를 분석하고 인식
- 각 재료의 신뢰도 점수를 제공하여 정확도 확인 가능
- 인식된 재료를 선택/해제하거나 **직접 추가**하여 커스터마이징

### 다양한 국가별 요리 선택

사용자가 원하는 국가의 요리 스타일을 선택할 수 있습니다:

- **한식** - 김치찌개, 불고기, 잡채 등
- **중식** - 마파두부, 탕수육, 볶음밥 등
- **일식** - 라멘, 돈부리, 규동 등
- **양식** - 파스타, 스테이크, 샐러드 등
- **이탈리아식** - 리조또, 까르보나라 등
- **태국식** - 팟타이, 똠얌꿍 등
- **베트남식** - 쌀국수, 반미 등
- **멕시코식** - 타코, 부리또 등
- **인도식** - 카레, 탄두리 치킨 등

### 17가지 테마별 레시피 생성

**시간/편의성 기반**

- 5분 준비 음식 - 바쁜 아침이나 야식용
- 초보자도 실패 없는 - 요리 경험 없어도 OK
- 5가지 이하 재료 - 간단하게 만드는 레시피

**건강/식단 기반**

- 단백질 보충 음식 - 운동 후 고단백
- 다이어트 저칼로리 - 칼로리는 낮추고 영양은 챙기기
- 탄단지 균형 건강식 - 영양소 밸런스
- 채식 위주 - 고기 없이도 맛있는 레시피

**장비/조리법 기반**

- 전자레인지만 있으면 되는 - 불 없이 간편하게
- 프라이팬 하나로 - 원팬 레시피
- 설거지 최소화 - 그릇 1~2개로 끝
- 냄새 적게 나는 조리법 - 원룸/오피스텔 친화적

**상황/형태 기반**

- 도시락/밀프렙 - 미리 만들어 보관
- 술안주/홈술 - 맥주/소주 한 잔에 어울리는
- 시원한 요리 / 뜨끈한 요리 - 날씨에 맞춘
- 아침 공복에 부담 없는 - 가볍게 먹는 아침식사
- 집들이용 간식/디저트 - 손님 접대용

### 조리 도구 필터링

보유한 조리 기구에 맞춰 레시피 생성:

- 전자레인지, 가스레인지/인덕션, 오븐
- 에어프라이어, 전기밥솥, 토스터기
- 믹서기/블렌더, 불 사용 안함(노쿡)

### AI 실시간 레시피 생성

- **고정된 레시피 DB가 아닌 AI가 실시간으로 생성**
- 재료 + 국적 + 테마 + 조리 도구를 조합하여 **맞춤형 레시피** 제공
- 상세한 조리 방법, 영양 정보, 조리 시간, 난이도 포함

## 기술 스택

### Frontend

- **Next.js 16** - React 프레임워크 (App Router, Turbopack)
- **React 19** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Tailwind CSS** - 유틸리티 기반 스타일링

### UI Components

- **Radix UI** - 접근성이 뛰어난 UI 프리미티브
- **shadcn/ui** - 재사용 가능한 컴포넌트 시스템
- **Lucide React** - 아이콘 라이브러리

### AI & Backend

- **Google Generative AI** - 이미지 기반 재료 인식
- **Supabase** - 백엔드 및 데이터베이스

### Development Tools

- **ESLint** - 코드 품질 관리
- **Prettier** - 코드 포맷팅
- **Vercel Analytics** - 성능 모니터링

## 사용 방법

1. **재료 사진 업로드**
   - 랜딩 페이지에서 "시작하기" 클릭
   - 냉장고 사진을 업로드하거나 카메라로 직접 촬영

2. **재료 확인 및 커스터마이징**
   - AI가 자동으로 인식한 재료 목록 확인
   - 신뢰도가 높은 재료(80% 이상)는 자동 선택
   - 잘못 인식된 재료는 선택 해제
   - 필요한 재료를 직접 추가 (카테고리 선택 가능)

3. **레시피 조건 선택**
   - **국가/요리 스타일**: 한식, 중식, 일식, 양식 등 9개 국가 중 선택
   - **테마**: 17가지 테마 중 원하는 스타일 선택 (빠른 요리, 다이어트, 채식 등)
   - **조리 도구**: 보유한 조리 기구 선택 (전자레인지, 에어프라이어 등)

4. **AI 레시피 생성**
   - "레시피 생성하기" 버튼 클릭
   - AI가 맞춤형 레시피 생성

5. **레시피 확인 및 요리**
   - 생성된 레시피 목록 확인
   - 각 레시피 카드 클릭하여 상세 정보 확인
   - 재료 목록, 조리 단계, 영양 정보, 조리 시간 등 포함
   - 마음에 드는 레시피를 저장하거나 공유 가능
